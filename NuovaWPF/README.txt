ABBIGLIAMENTO è UN'APPLICAZIONE WPF Windows Presentation Foundation (WPF), un framework dell'interfaccia utente 
indipendente dalla risoluzione e che usa un motore di rendering basato su vettori, 
progettato per sfruttare i vantaggi dell'hardware grafico moderno. WPF offre un set completo di funzionalità per lo sviluppo 
di applicazioni che includono Extensible Application Markup Language (XAML), controlli, data binding, layout, grafica 2D e 3D, 
animazioni, stili, modelli, documenti, supporti, testo e tipografia. WPF fa parte di .NET e, 
pertanto, è possibile compilare applicazioni che incorporano altri elementi dell'API .NET.





Un sistema di gestione negozio di abbigliamento. 

Sviluppare un sistema di gestione per un negozio di abiti che permetta la gestione di prodotti con variazioni di colore e taglia, categorie di prodotti, utenti e ordini. 

Componenti del Sistema: 

Gestione Prodotti:  

Aggiungere, modificare, eliminare prodotti. 

Ogni prodotto ha una galleria di prodotti (potete usare dei link ad immagini esterne, tipo Lorem Picsum). 

Gestire le variazioni di ogni prodotto (colore, taglia). 

Ogni prodotto ha una combinazione di variazioni ed una quantità ed un prezzo per ogni variazione (guarda la tabella sotto per un esempio, è parziale ma serve a rendere l'idea). 

Creare ordini associati a un utente. 

Includere prodotti con le specifiche variazioni in un ordine. 

Visualizzare e gestire lo stato degli ordini. 

Il sistema dovrà essere gestito, se possibile con delle viste diverse create in WPF, integrare una gestione menu in alto alla pagina per cambiare le varie visualizzazioni (File->Gestione …)
